# Projeto-C21

Tarefas específicas para completar o projeto:
1. Altere os valores da tela e aumente o tamanho da tela.
2. Crie uma variável, por exemplo, ball (bola).
3. Em setup(), crie uma variável balls_option (opções da bola) e atribua propriedades.
  ● Certifique-se de definir a densidade (density) da bola de papel para 1.2,
conforme mostrado abaixo.
4. O objeto bola será o objeto em movimento no jogo. 
  ● Você tem que criar um corpo circular para a bola e adicioná-lo ao World.
5. Crie um projeto para o Ground (solo), sobre o qual fica a lata de lixo e também onde a
bola de papel amassada poderá cair.
  ● Para exibir o solo na tela, crie um display() na classe ground.js.
  ● Chame display() dentro de draw().
6. Para criar uma cesta, crie mais dois objetos para a classe de solo, nesse caso, leftSide
e rightSide.
  ● Chame display() de ground.js com as classes leftSide e rightSide.
7. Faça a bola de papel amassado pular e cair na cesta com a tecla de seta para cima.

8. Certifique-se de que o projeto funciona antes de enviá-lo.
